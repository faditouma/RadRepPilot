# ACR Extraction Quality Reports

This folder is for local/developer review outputs from the ACR PDF extraction pipeline.

The generated CSV files can be opened in Excel, Apple Numbers, Google Sheets, or another spreadsheet tool. They are intended to help identify extraction errors before any topic is transformed, reviewed, and made public in RadRepPilot.

These reports do not contain full PDF text. Procedure row previews are intentionally short and should be used only to guide manual review against the original local source PDFs.

Generated CSV files are ignored by Git by default. Keep this README tracked, and only commit generated reports if you intentionally want to share a review snapshot.
